Enter the secret code in the proper sequence to set off an image that will slide across your browser screen along with the tune of your favorite sound!

== Description ==

Trigger an image to slide across your browser screen to your desired tune by entering the correct secret code on your keyboard.

You could keep this easter egg a secret and let users find out about this hidden gem on their own, *or* you could prank someone's computer by turning their volume all the way up and ask them to enter in the code.

The choice is yours!

Accepted image types: **.JPG**, **.JPEG**, **.PNG**
Accepted tune types: **.MP3**

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress 'Plugins' screen directly.
1. Activate the plugin through the WordPress 'Plugins' screen.
1. Click on the Settings link underneath the plugin name within the WordPress 'Plugins' screen or click on the plugin name within the main left-hand WordPress menu to be taken to the plugin's settings screen.
1. Use the plugin settings screen to configure your secret image and tune options.
1. Make sure to save any changes you made within the plugin settings screen.

== Frequently Asked Questions ==

= What type of images can I select? =
You can only use images that have a **.JPG**, **.JPEG**, or a **.PNG** extension

= What type of tunes can I pick? = 
You can only use tunes that have a **.MP3** extension

== Screenshots ==

1. Main title, subtitle, and secret code input field. Users can enter in their own secret code is to trigger the image slide and tune.

2. Choose an image you would like to have sliding across your screen. Images with a **.JPG**, **.JPEG**, or **.PNG** extension are only accepted.

3. Choose a tune you would like to have playing when the secret image is sliding across your screen. Tunes with a **.MP3** extension are only accepted.

4. Enter a positive or negative number to define the starting position for your secret image.

5. Enter a positive or negative number to define the ending position for your secret image.

6. Enter a number greater than 0 to define how fast (or slow) the secret image will slide across your browser screen.
